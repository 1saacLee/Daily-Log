# GAMES101-现代图形学基础

本系列是闫令琪的GAMES101课程笔记，由于时间比较长，而且很多东西是纸质版笔记，因此这里可能只做简单记录

GAMES101是入门图形学的第一门课，个人认为讲解浅显易懂，而且闫老师幽默风趣，是很好的入门课程。目前计划的主要笔记内容包括变换、投影、光栅化、着色、几何等。由于部分普及性的视频我没看过，因此例如光线追踪等知识就先不记录了
